# Artem Krokhinov — Personal Portfolio

A modern, high-performance portfolio website designed to showcase my skills in Full-Stack development and system architecture.

## Key Features

- **Editorial Design:** A unique, sophisticated style with an emphasis on typography (Inter and Playfair Display fonts) and minimalism.
- **Responsive Grid:** A complex CSS Grid layout for project presentation (3 rows by 2 columns) that smartly adapts to mobile devices.
- **Interactivity:** Smooth hover effects and micro-animations that enhance the user experience without sacrificing performance.
- **Tech Stack:** React, Vite, and Vanilla CSS (avoiding heavy UI libraries to ensure total control over styling and maximum load speed).
